# 세션 무효화 공지문 (템플릿)

제목: 보안 강화 업데이트 안내 (재로그인 필요)

안녕하세요. SolveMate 팀입니다.

서비스 보안을 강화하기 위해 인증 시크릿(`JWT_SECRET`)을 교체하였습니다.  
이 조치로 인해 기존 로그인 세션이 만료되어 앱 재로그인이 필요합니다.

## 적용 일시

- YYYY-MM-DD HH:mm (KST)

## 영향 범위

- 기존 로그인 세션 자동 로그아웃
- 재로그인 후 기존 데이터(연결 정보/목표/메모/추천)는 그대로 유지

## 사용자 안내

1. 앱을 최신 버전으로 업데이트해 주세요.
2. 다시 로그인해 주세요.
3. 로그인 문제가 지속되면 아래 문의처로 연락해 주세요.

## 문의처
